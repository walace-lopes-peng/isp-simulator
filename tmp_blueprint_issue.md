Este documento será a 'Constituição' do ISP Tycoon, definindo como o jogo deve evoluir de um protótipo para um simulador completo. O arquivo PROJECT_BLUEPRINT.md na raiz do projeto servirá como âncora para a lógica de desenvolvimento.

Pede-se para referenciar `PROJECT_BLUEPRINT.md` para:
1. Visão Geral (The North Star)
2. O Motor de Fluxo (Core Loop)
3. Progressão de Eras (Historical Evolution)
4. O Motor de Cenários (Modular Engine)
5. Regras Técnicas de Desenvolvimento (Guardrails)
